Simple Mode

中文智能研究助手

直接用自然语言说你想看的股票，例如“今天看看 600519 怎么样”。 助手会自动补最近 K 线、生成技术因子，并给出综合分、60/120 日视角、四个 Teacher 模型分项和简短理由。

边界提醒
  • 只有明确要求查看具体股票时才会联网补行情。
  • 单股评分会自动从 K 线生成技术因子。
  • 评分是研究辅助，不构成投资建议。

Online Kline

显式下载 A 股 K 线

只有点击启动后才会联网。普通模式默认拉取近 120 交易日左右的 K 线，并在解释中同时给出近 60 日与近 120 日两个评分视角。 依赖服务器网络、akshare、tushare 与 TUSHARE_TOKEN；不可用时仍可上传本地数据。

Simple Scoring

结构化新信号评分流程

上传或粘贴候选信号后，系统会先校验 signal_id、signal_date、symbol、signal_type 和结构化 features。 旧字段 date 可以作为 signal_date 的别名自动识别。 默认先做 prompt preview / dry run / mock / 历史样本复核校验；当本地 GPT-OSS runtime 健康时，也可以切换到 live scoring。

Local GPT-OSS runtime offline

当前未连接到本地 GPT-OSS。你仍可使用 prompt_only / dry_run / mock / 历史样本复核； live scoring 需要先启动本地 vLLM 服务。任何 scoring 都不构成投资建议。

                    This page is the main product entry for new research work. It walks from task classification to data

Run & Monitor

                  
Create artifacts automatically, launch the workflow, and inspect the canonical execution chain.
